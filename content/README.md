# Blog posts

Add your own blog posts here as **`.mdx`** files.

## Format

Each post needs:

1. **Filename** – Use kebab-case, e.g. `my-first-post.mdx`. The filename (without `.mdx`) becomes the URL slug: `/blog/my-first-post`.

2. **Frontmatter** (at the top, between `---`):

```yaml
---
title: "Your Post Title"
publishedAt: "2025-02-25"
updatedAt: "2025-02-25"   # optional
author: "Your Name"        # optional
summary: "Short description for the list and SEO."
image: "/path/to/image.jpg"  # optional
---
```

3. **Body** – Write your content in Markdown/MDX below the frontmatter.
